About this Game

This game is designed by using HTML,CSS and javascript functionalities. The game contains all the functionalities of a Simon Game. The computer is programmed to pick colours from one of the given colours(Red,Green,Yellow,Blue) in a pattern, as the game progress if also checks what colour is picked by the user, and what is the pattern of the user colour choices. Each of the four colour acts as a button when a user clicks a buttons it is detected by the computer by which the computer decides what action to take next by running it with the given code.

How to play the Simon Game?

Press any key on the QWERTY keypad to start the game, the computer will pick a colour[Red], then click on the same button [Red] now the computer will check if you pick the right colour or not. If you pick the right colour(same as computer) then the computer will pick the next colour [green], now you have to memorise the pattern in which the colours has been picked and the press the buttons in the same pattern as the computer[red, green]

for example:

computer = [Red] then you = [Red]

computer = [Green] then you = [Red,Green]

with each user click the computer will check the pattern with computer pattern. If the pattern match the game will continue, if the user click the wrong button the game will be over and the user has to restart the game. with each correctly guessed pattern the level will increase.

The Website contains a clock which also acts as stop-watch.

If you click the start button the stopwatch starts and when you click the stop button it stops the can be restarted from the previous time with the start button.

If the user click on the watch if starts to functions as a normal digital clock and resets the stopwatch
